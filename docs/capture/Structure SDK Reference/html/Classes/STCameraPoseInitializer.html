initWithVolumeSizeInMeters:options:error:

Initialize with all the required fields.

- (instancetype)initWithVolumeSizeInMeters:(GLKVector3)volumeSize options:(NSDictionary *)options error:(NSError *__autoreleasing *)error

Parameters

volumeSize

The current volume size in meters. Not used by the STCameraPoseInitializerStrategyGravityAlignedAtOrigin strategy.

options

The option dictionary. The valid keys are:

  • kSTCameraPoseInitializerStrategyKey: STCameraPoseInitializerStrategy value to specify the estimation strategy. Required.
error

will contain detailed information if the provided options are incorrect.

updateCameraPoseWithGravity:depthFrame:error:

Update the current pose estimate from a depth frame and a CoreMotion gravity vector, using the strategy specified at init.

- (BOOL)updateCameraPoseWithGravity:(GLKVector3)gravity depthFrame:(STDepthFrame *)depthFrame error:(NSError *__autoreleasing *)error

Parameters

gravity

a gravity vector in IMU coordinates, e.g. as provided by CMDeviceMotion.gravity.

depthFrame

the current processed depth from Structure Sensor. Can be nil if using the strategies STCameraPoseInitializerStrategyGravityAlignedAtOrigin or STCameraPoseInitializerStrategyGravityAlignedAtVolumeCenter.

error

will contain detailed information if the estimation failed.

Return Value

TRUE on success, FALSE on failure.

detectInnerPixelsWithDepthFrame:mask:

Determine which pixels of a depth frame are inside the current scanning volume.

- (void)detectInnerPixelsWithDepthFrame:(STDepthFrame *)depthFrame mask:(uint8_t *)outputMask

Parameters

depthFrame

the current processed depth from Structure Sensor.

outputMask

should point to an allocated buffer of (depthFrame.width * depthFrame.height) pixels. A mask pixel will be 255 if the corresponding 3D point fits inside the volume, 0 if outside or if there is no depth.
